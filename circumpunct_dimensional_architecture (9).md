## 1. Foundational Principles

### 1.1 The Circumpunct

The fundamental object is the circumpunct ⊙, an irreducible trinity:

```
⊙ = • ⊗ Φ ⊗ ○
```

| Symbol | Name | Role |
|:------:|:-----|:-----|
| • | Aperture / Center / Soul | Where transformation occurs |
| Φ | Field / Mind | Mediates between center and boundary |
| ○ | Boundary / Body / Surface | Interface with exterior |

Center and boundary cannot interact directly — field mediation is geometrically necessary.

### 1.2 The Master Equation

Field evolution proceeds through three stages:

```
Φ' = ⊱ ∘ i ∘ ≺[Φ]
```

| Operator | Name | Action |
|:--------:|:-----|:-------|
| ≺ | Convergence | Gathers field toward aperture |
| i | Aperture rotation | Transforms at the center (90° at balance) |
| ⊱ | Emergence | Expresses transformed field outward |

**Expanded form:**

```
Φ'(x) = ∫ dy K_out(x,y) · i · ∫ dz K_in(y,z) Φ(z)
```

### 1.3 Operator Notation

**Adjoint pairs:**
- (≻, ≺) — convergence pair
- (⊰, ⊱) — emergence pair

**Convention:** The open (convergent) side of each arrow points toward the aperture.

**Standard triple:** (≺, i, ⊱)

```
Φ' = ⊱ ∘ i ∘ ≺[Φ]

Read compositionally (right-to-left):
    ≺ acts first   (convergence)
    i acts second  (transformation)
    ⊱ acts last    (emergence)
```

## 2. Dimensional Ontology

### 2.1 Two Kinds of Dimension

| Type | What it is | Nature |
|:-----|:-----------|:-------|
| **Integer** | Structure / Product | What time creates (the braid) |
| **Fractal** | Process / Time | The braiding itself |

```
Fractal = braiding (time, process, verb)
Integer = braid (structure, product, noun)
```

Integer dimensions are the **artifacts** of temporal process — stable woven objects.
Fractal dimensions are **time itself** — the becoming, the weaving, the recursion.

**Note:** When we write "3.5D," we mean degree-of-process between 3 and 4 — not a new orthogonal coordinate axis.

### 2.2 Half-Integers as Convention

The half-integer dimensions (0.5, 1.5, 2.5...) are the simplest markers of "in-between-ness." But time is really the **entire fractional interval** (n, n+1) — a spectrum of temporal depth, not just discrete 0.5 steps.

This matches the fractal surface: time lives in the nested in-betweens.

### 2.3 The Mandelbrot Foundation

Fractional (Hausdorff) dimensions are rigorously established mathematics:

- Hausdorff dimension formalism is standard measure theory
- Brownian motion's D = 1.5 is a theorem (not a fit)
- Non-integer dimensions are not controversial

The specific D value varies by system — this variation is expected:
- Coastlines ≈ 1.25
- Brownian motion = 1.5 (exactly)
- DLA clusters ≈ 1.7

The circumpunct IS consciousness. There is no separate "consciousness dimension."

```
⊙ = • ⊗ Φ ⊗ ○ = consciousness = wholeness
```

Consciousness isn't located in a dimension — it's what the circumpunct *is*.

| Component | Information Type | What it holds |
|:---------:|:-----------------|:--------------|

| **Φ** | Analog | Complex amplitude (continuous, includes phase) |

**Phase and gating are not independent.** They are the same coherence distinction seen through different components:
- Through Φ → phase (continuous interference)
- Through • → bit (threshold decision)  
- Through ○ → nested gating (fractal repetition at every scale)

### 8.4 The 64 States

The **64 states** are the center's 6-bit coherence signature:

```
s = (b_•r, b_Φr, b_○r, b_•i, b_Φi, b_○i) ∈ {0,1}⁶

6 channels:  (•, Φ, ○) × (real, imaginary)
Each channel: binary coherence bit (0 or 1)
Total states: 2⁶ = 64
```

The field holds analog amplitudes. The center compresses that analog reality into a binary decision per channel. The boundary fractalizes it across scales.

**Note on ○ bits:** Since ○ is fractal (many nested gates), what does a single bit represent? The ○ bits are the center's coherence verdict about the boundary's current effective transmission — a coarse-grained summary of many nested gates.

**Convergence and emergence always happen.** The 64 states encode *which channels achieve coherence*, not which direction they flow.
